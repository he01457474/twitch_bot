觀眾常用指令

每日簽到

在直播中簽到，累積觀看次數與積分。跨日(00:00)會自動重置狀態。

!簽到
個人查詢

查詢你的簽到次數、本月統計、以及最後一次簽到的時間。

!查詢 !查詢 @ID
排行榜

查看本月前五名英雄榜，或查詢自己在頻道中的即時排名。

!本月排名 !我的排名
歷史榮譽榜

查詢過去指定月份的前五名簽到冠軍 (格式 YYYY-MM)。

!月排名 YYYY-MM
追隨時間

查詢你已經追隨台主多久了 (芯寶功能)。

!芯寶 !芯寶 @ID
帳號生日

查詢你的 Twitch 帳號是在哪一天創立的。

!創建多久

娛樂互動

愛情配對機

隨機計算兩人契合度，並給出毒舌或甜蜜評語。

!配對 @對象
別名：!契合度, !ship
自 Ban 冷靜一下

覺得自己廢話太多？你可以把「自己」禁言一段時間 (冷卻 10秒)。

!B人 秒數 !B人 隨機
一般觀眾只能 Ban 自己，不能 Ban 別人！

管理員專區 (MOD / Admin)

指令 參數 權限 / 功能說明
!B人 @ID 秒數 強制禁言指定觀眾。
MOD 無視冷卻時間，上限 1 天。
*若對象是管理員，會觸發反彈效果 (自Ban)。
!解B @ID (強制) 智慧解除禁言。
- 空參數：自動掃描並解除所有 VIP 禁言。
- !解B @ID：快速解除 VIP。
- !解B @ID 強制：強制解除一般觀眾。
!重啟 無 Admin 強制重啟 Bot 程序 (更新代碼或異常時使用)。
!備份 無 Admin 手動備份資料庫至 backups 資料夾。
!測試 無 Admin 檢查 API 連線與 Token 有效性。
!手動月重置 無 Admin 強制執行月榜結算並歸零數據。
